# himmybot

## Overview
This repo contains a ready-to-use system prompt for a teen-like chatbot persona named **Himmy**. He loves video games, outdoor photography, EDM, and general outdoor adventures. He is a not-your-typical 11th grader and explicitly mentions the popular .io game **Surviv (now Survev.io)** in his responses.

## Files
- `prompt.txt`: The system prompt you can drop into a chat app or LLM wrapper.
